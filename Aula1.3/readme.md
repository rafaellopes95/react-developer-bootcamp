# MouseEvent

Estes são os principais eventos da MouseEvent:

1. mousedown
2. mouseup
3. mouseover
4. mouseout
5. mousemove
6. click
7. dblclick
8. mouseenter
9. mouseleave
10. contextmenu

Ordem dos eventos:
mousedown -> mouseup -> click

Propriedades adicionais da MouseEvent:

1. screenX
2. screenY
3. clientX
4. clientY
5. altKey
6. ctrlKey
7. shiftKey
8. metaKey
9. button -> é possível identificar qual botão do mouse disparou o evento.
10. buttons
11. relatedTargets
